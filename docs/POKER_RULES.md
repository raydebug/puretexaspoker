# 德州扑克 (Texas Hold'em) 游戏规则手册

欢迎使用本测试系统内置的德州扑克规则指南。德州扑克是目前世界上最流行的扑克衍生游戏，结合了策略、心理和概率。

## 1. 游戏目标
利用手中的 **2张底牌** 和桌面上的 **5张公共牌**，组合出最强的 **5张牌** 手牌，以赢得奖池（Pot）。

## 2. 基础设置
- **庄家 (Dealer/BTN)**：每局由庄家位开始顺时针转动。本测试系统中庄家由系统自动指派。
- **小盲注 (Small Blind/SB)**：位于庄家左手边第一位，先行投入强制注。
- **大盲注 (Big Blind/BB)**：位于庄家左手边第二位，通常是小盲注的两倍。

## 3. 游戏轮次
每局游戏分为四个发牌阶段，每个阶段后都有一次投注环节：

1.  **翻牌前 (Pre-flop)**：每位玩家发放 2 张不公开的底牌。
2.  **翻牌 (Flop)**：桌面发放 3 张公开的公共牌。
3.  **转牌 (Turn)**：桌面发放第 4 张公共牌。
4.  **河牌 (River)**：桌面发放第 5 张公共牌。

## 4. 玩家动作
在轮到自己操作时，玩家可以选择：
- **弃牌 (Fold)**：放弃底牌和当前投入，退出本局。
- **过牌 (Check)**：不加注，直接交由下一位玩家（仅在当前轮没有人下注时可用）。
- **跟注 (Call)**：投入与当前最高加注额相等的筹码。
- **加注 (Raise)**：增加当前注额，强制其他玩家投入更多筹码或弃牌。
- **全下 (All-in)**：投入所有剩余筹码。

## 5. 牌型面值 (由大到小)

| 牌型 | 描述 | 示例 |
| :--- | :--- | :--- |
| **皇家同花顺** | A-K-Q-J-10 且花色相同 | A♠ K♠ Q♠ J♠ 10♠ |
| **同花顺** | 五张连号且花色相同 | 9♥ 8♥ 7♥ 6♥ 5♥ |
| **四条** | 四张面值相同的牌 | A♦ A♣ A♥ A♠ 5♣ |
| **葫芦** | 三条 + 一对 | K♠ K♥ K♦ 2♣ 2♥ |
| **同花** | 五张花色相同但非连号 | A♣ J♣ 8♣ 4♣ 2♣ |
| **顺子** | 五张连号但花色不同 | 5♠ 4♥ 3♦ 2♣ A♥ |
| **三条** | 三张面值相同的牌 | 9♦ 9♣ 9♠ K♥ 2♦ |
| **两对** | 两个不同的对子 | Q♠ Q♦ J♣ J♥ 5♠ |
| **一对** | 两个面值相同的牌 | 10♦ 10♣ A♠ K♥ 4♦ |
| **高牌** | 没有任何组合，比面值 | A♠ (A高牌) |

## 6. 摊牌与胜负 (Showdown)
如果最后一次下注结束后仍有多于一名玩家在局中，则进行摊牌。拥有最强五张手牌组合的玩家赢得奖池。如果牌型完全相同，则平分奖池（Split Pot）。
